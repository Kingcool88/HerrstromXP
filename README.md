# FamilyQuest V7.1

Familjeapp med Google-inloggning, barnläge som standard och adminläge bakom PIN.

## Nytt i V7.1

- Nytt neutralt namn: FamilyQuest
- Landningssida innan inloggning
- Startup-guide första gången
- Skapa familj, tema, admin-PIN och barnprofiler
- Barnläge/familjeläge som standard
- Föräldra-admin separat bakom PIN
- Nintendo/speltema och stilrent tema
- Kompisliga/League Points, statistik, uppdrag och belöningar
